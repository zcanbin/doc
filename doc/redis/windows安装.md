### 1、下载

- 直接点击这里：https://github.com/tporadowski/redis/releases，选择最新版下载

### 2、解压

解压到合适的目录，并进入到redis目录

### 3、安装（可选）

- 安装成系统服务命令

```shell
redis-server --service-install redis.windows-service.conf --loglevel notice --service-name redis
```

- 删除服务的命令为：

```shell
redis-server --service-uninstall  
```
